O Jarila! Ads é um plugin para gerenciar e facilitar a exibição de anúncios do Mercado Livre e Google AdSense (mais redes virão).


== Description ==

O Jarila! Ads é um plugin para WordPress exclusivamente desenvolvido para ajudar a monetizar o seu website, inserindo ofertas de um ou mais
parceiros nos posts que você publica. Este plugin torna fácil a inserção de publicidade segmentada pois é você, editor do post, que sabe
exatamente quais as melhores ofertas e anúncios para aquele post, diferente de tecnologias como o Google AdSense, que tenta adivinhar
qual o melhor anúncio para o post baseado no texto que você publicou. A eficiência do Jarila! Ads em muitos casos é muito superior ao
Google AdSense. Sem contar que vários parceiros (anunciantes) podem ser controlados à partir do Jarila! Ads, que podem incluir o próprio
AdSense.

== Installation ==

1. Unzip the contents of this ZIP file into a folder called "wp-jarila" under the WordPress' "\wp-content\plugins\" folder.
2. Activate the plugin through the  "Plugins" manu in WordPress
3. Go to the  "Options" section of your WordPress administration, and choose  "Jarila! Ads". Set all the properties.
4. Write a post. You will notice that a special section named Jarila! Ads will appear on it. Enjoy!
